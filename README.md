# BusinessCard App 💼

A simple Android application that displays a digital business card with a clean and professional UI.

## 🚀 Features
- Digital business card display
- Clean UI design
- Contact information layout

## 🛠️ Tech Stack
- Kotlin
- Android Studio
- XML Layouts

## ▶️ How to Run
1. Clone the repository
2. Open in Android Studio
3. Run the app
